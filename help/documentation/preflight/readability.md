# Preflight 가독성 기회

![Preflight 가독성 기회](./assets/readability/hero.png){align="center"}

Sites Optimizer의 Preflight 가독성 기회를 통해 웹 사이트 콘텐츠의 가독성과 관련된 문제를 식별하고 해결할 수 있습니다. 이 기회는 사이트의 콘텐츠 구조를 평가하는 데 중점을 두며 개선을 위해 실행 가능한 권장 사항을 제공합니다.

Preflight 가독성 기회는 다음과 같은 여러 언어로 제공됩니다.

* 영어
* 프랑스어
* 스페인어
* 이탈리아어
* 독일어
* 네덜란드어

이를 통해 팀은 다양한 대상자에서 콘텐츠 가독성을 평가하고 최적화할 수 있습니다.
